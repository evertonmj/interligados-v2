# ⚡ Quick Start - Interligado 3 Modos

## 🎯 Comece AGORA em 3 Passos

### Opção 1: MAIS RÁPIDO (Offline - Recomendado)
```
1. Abra o app
2. Clique em "Offline"  
3. Clique em um tema (ex: "Tecnologia")
4. Clique "Iniciar Jogo"
✅ Pronto! Grafo carrega instantaneamente
```

---

### Opção 2: MAIS CRIATIVO (Gemini IA)
```
1. Acesse: https://makersuite.google.com/app/apikey
2. Copie sua API Key
3. No app, clique em "Gemini IA"
4. Cole a API Key
5. Digite qualquer tema (ex: "Culinária")
6. Clique "Iniciar Jogo"
✅ Aguarde 2-5 segundos
```

---

### Opção 3: MAIS PRIVADO (Ollama Local)
```
1. Instale: https://ollama.ai
2. Rode no terminal: ollama pull mistral
3. Rode: ollama serve
4. No app, clique em "Ollama Local"
5. Digite qualquer tema
6. Clique "Iniciar Jogo"
✅ Aguarde 5-30 segundos (depende do seu PC)
```

---

## 📊 Qual Modo Escolher?

**Quer RAPIDEZ?** → 🟢 **Offline**  
**Quer CRIATIVIDADE?** → 🔵 **Gemini**  
**Quer PRIVACIDADE?** → 🟠 **Ollama**

## 🎮 Como Jogar

1. **Escolha um tema** (palavra inicial)
2. **Veja o grafo** com palavras conectadas
3. **Adivinhe as palavras ocultas** (aparecem como `?`)
4. **Ganhe revelando todas** as palavras

### Dicas de Jogo
- 💡 Leia as palavras visíveis para pistas
- 📈 Número maior = mais difícil
- 🚀 Offline = mais rápido para testar
